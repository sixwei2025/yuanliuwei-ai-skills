# 用AI验证需求，不花钱不走弯路

核心理念：**大多数创业失败不是因为执行差，而是因为在一个不存在的需求上认真执行。** 验证需求的成本应该接近于零。

---

## 验证框架：PACT模型

### P — Pain（痛点真实性）
这个痛点是「止痛药」还是「维生素」？

- **止痛药级别**，不解决会让人睡不着觉，客户会主动找你
- **维生素级别**，有了更好没有也行，需要你主动去推销

验证方式，在目标用户聚集的地方（微信群、知乎、小红书、行业论坛）搜索相关话题，看有没有人在主动提问或抱怨这个问题。

### A — Audience（受众可触达性）
你的目标客户在哪里，你能到达吗？

- 他们聚集在哪些平台？微信群/知乎/小红书/行业社群
- 你有没有进入这些圈子的路径？
- 触达第一批10个潜在客户需要多少成本和时间？

### C — Competition（竞争格局判断）
有竞争是好事，没竞争才危险（说明没市场）。

竞品分析三问：
1. 现有竞品用什么方式解决这个问题？
2. 他们的定价是多少？用户反馈如何？
3. 你能在哪个维度比他们好10%以上？（不需要全面碾压，一个维度够了）

### T — Transaction（付费意愿验证）
有人有痛点不等于有人愿意付钱。

最直接的验证，让用户付押金预订，或者找到3个人愿意当种子客户。**口头说"这个很好"不算验证，付钱才算。**

---

## 第一步：拆解用户的想法

收到用户的业务方向后，先做结构化拆解：

```
你想做的是：[一句话描述]
目标客户是：[具体的人，不是泛指]
解决的问题是：[具体的痛点]
收费方式是：[按次/按月/按项目]
竞品是：[现在人们怎么解决这个问题]
```

如果用户描述模糊，主动追问直到以上5项清晰。

---

## 第二步：AI辅助竞品研究

用web_search工具执行以下搜索（必须实际搜索，不能靠记忆）：

**搜索清单：**
1. `[核心关键词] + 多少钱` / `[核心关键词] + 价格` — 了解市场定价
2. `[核心关键词] + 小红书` — 看C端用户真实讨论
3. `[核心关键词] + 知乎` — 看深度需求讨论
4. `[竞品名称] + 评价` / `[竞品名称] + 怎么样` — 看用户反馈
5. `[核心关键词] + 招聘` — 有企业在招相关岗位说明需求真实存在

搜索完后，整理成竞品矩阵：

| 竞品/替代方案 | 定价 | 优点 | 用户抱怨 | 你的切入点 |
|------------|------|------|---------|---------|
| [竞品A] | | | | |
| [竞品B] | | | | |
| 现有替代方案 | | | | |

## 第三步：付费意愿测试方案

根据用户的业务类型，给出定制化的测试方案：

### B端业务（企业客户）
**最小验证动作：** 约5家目标企业的负责人做30分钟访谈（不是推销，是了解现状）
- 访谈核心问题，你们现在怎么解决[这个问题]？一年花多少钱？最头疼的是什么？
- 验证标准，如果3家以上表示「如果价格合适愿意试用」，说明需求存在

### C端业务（个人用户）
**最小验证动作：** 在目标用户社群发一条「征集内测用户」的帖子
- 帖子内容，我在做[具体产品]，免费给[N]个人内测，要求是[简单门槛]，感兴趣的留言
- 验证标准，48小时内有10人以上真实响应（不是泛泛点赞）

### 知识付费/培训
**最小验证动作：** 发布一个「预售」或「拼团」活动
- 定价是最终价格的5折，设置人数门槛（比如满20人开班）
- 验证标准，72小时内达到门槛人数

## 第四步：输出验证报告

```
## 需求验证报告：[业务名称]

### 竞品研究结论
[3-5句话总结市场现状，有没有竞品、定价区间、用户痛点]

### 需求真实性判断
评级：🔴 不推荐 / 🟡 待观察 / 🟢 值得做

判断依据：
- 市场有没有人在为类似问题付钱？[是/否 + 证据]
- 竞品的用户反馈揭示了什么未满足的需求？[具体内容]
- 你的差异化空间在哪里？[具体描述]

### 你的验证行动计划
第1-7天：[具体行动]
第8-14天：[具体行动]
第15-30天：[具体行动]

### 验证成功的标志
[明确说明什么结果代表需求验证通过，可以投入资源]

### 验证失败的信号
[明确说明看到什么迹象应该转方向，不要恋战]
```

---

## 关键原则

**验证期不要做产品。** 验证期的目标是找到愿意付钱的人，不是做一个完美产品。能用表格、微信群、手动操作交付的，就先用这些，等验证通过了再考虑自动化。

**拒绝和沉默都是信息。** 如果发了验证帖没人响应，这不是坏消息，这是有价值的数据，说明触达方式或产品定位需要调整。

**不要问「你觉得这个好不好」。** 这种问题得不到真实答案，要问「你现在怎么解决这个问题，花了多少钱和时间」。
