# 一人AI公司 Skills 包

> 基于袁六伟「一人AI公司」实战方法论，打包为10个可调用的AI Skill

---

## 关于这套Skill

这不是理论，这是一套跑通过的操作系统。

袁六伟用这套方法论，**一个人**服务了80+企业客户、4000+学员，搭建了六智能体系统处理80%的日常事务，年收入突破百万。这套Skill把他的核心方法论拆解为10个可直接调用的决策工具。

---

## 包含的10个Skill

| Skill | 触发场景 | 核心输出 |
|-------|---------|---------|
| `find-your-niche` | 不知道做什么方向 | 3个最优赛道 + 验证路径 |
| `validate-with-ai` | 有想法不确定有没有市场 | 竞品分析 + 付费验证方案 |
| `price-your-service` | 不知道收多少钱 | 定价方案 + 报价话术 |
| `deliver-first-client` | 拿到第一个客户不知道怎么做 | 五阶段交付SOP |
| `build-your-agent-team` | 想搭多AI助手系统 | AI团队设计方案 |
| `content-flywheel` | 需要持续内容获客 | 四阶段内容流水线 |
| `build-your-community` | 想做私域和知识付费 | 三层变现结构 |
| `package-your-ip` | 想把经验变成产品 | IP产品矩阵设计 |
| `automate-your-ops` | 一个人搞不过来 | 自动化优先级地图 |
| `boss-level-thinking` | 决策迷茫或缺乏方向感 | CEO视角决策框架 |

## 安装方式

### 在 Claude.ai 中使用
将任意 `SKILL.md` 文件内容复制到 Claude Project 的 System Prompt 中即可。

### 在 Claude Code 中使用
```bash
# 将整个目录复制到你的skills文件夹
cp -r yiluren-ai-skills/ ~/.claude/skills/

# 或安装单个skill
cp -r yiluren-ai-skills/find-your-niche/ ~/.claude/skills/
```

### 使用 .skill 文件安装（如平台支持）
```
/skill install find-your-niche.skill
```

---

## 使用建议

**创业初期（0-3个月）**，优先使用 `find-your-niche` → `validate-with-ai` → `price-your-service`

**启动阶段（3-6个月）**，重点使用 `deliver-first-client` → `build-your-agent-team` → `content-flywheel`

**增长阶段（6个月+）**，深入使用 `build-your-community` → `package-your-ip` → `automate-your-ops`

**任何阶段**，`boss-level-thinking` 是底层操作系统，遇到迷茫随时调用。
